## 🤝 How to contribute

1. Fork the repo you want to contribute to.  
2. Create a feature branch: `git checkout -b feat/your-feature`.  
3. Commit changes and push.  
4. Open a Pull Request with a clear description of what you changed.  

We follow Git conventional‑commits and clean branching practices. Check the `CONTRIBUTING.md` file in each project.
